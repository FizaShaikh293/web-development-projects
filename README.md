# 💻 Web Development Projects

This repository contains multiple web development projects showcasing backend development, database integration, and full-stack application concepts.

---

## 📂 Projects Included

### 🏨 Hotel Management System (PHP)

A complete hotel management system developed using PHP for handling bookings, user management, and administrative operations.

#### 🔧 Features

* 🛏️ Room booking system
* 👤 User management
* 📊 Profit tracking
* 🔐 Login & logout functionality
* 🗂️ Structured backend with configuration and database handling

#### 📁 Structure

```bash
hotel-management-system/
│
├── assets/
├── config/
├── database/
├── user/
├── home.php
├── roombook.php
├── profit.php
├── logout.php
├── HOTELMANAGEMENT_Presentation.pdf
└── ProjectDocumentation.pdf
```

#### 🛠️ Tech Stack

* PHP
* MySQL
* HTML/CSS

---

### 🏢 Leave Management System (Node.js)

A backend-based leave management system that allows students to apply for leave and enables authorities to approve or reject requests.

#### 🔧 Features

* 👩‍🎓 Student leave requests
* 🧑‍💼 Role-based system (Student, HOD, Warden)
* ✅ Approval & rejection workflow
* ⚡ Lightweight Node.js backend

#### 📁 Structure

```bash
LeaveManagement-NodeJS/
│
├── models/
├── image/
├── app.js
├── package.json
├── package-lock.json
└── .gitignore
```

#### 🛠️ Tech Stack

* Node.js
* Express.js
* JavaScript

## 🚀 How to Use

### Clone the Repository

```bash
git clone https://github.com/FizaShaikh293/web-development-projects.git
cd web-development-projects
```

---

### ▶️ Run Leave Management System

```bash
cd LeaveManagement-NodeJS
npm install
node app.js
```

---

### ▶️ Run Hotel Management System

* Place the project inside **XAMPP / WAMP htdocs folder**
* Start Apache & MySQL
* Open in browser:

```
http://localhost/hotel-management-system
```
