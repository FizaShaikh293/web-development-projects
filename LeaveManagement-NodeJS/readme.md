# 🏢 Leave Management System (Node.js)

A web-based Leave Management System built using Node.js that allows students to apply for leave and enables authorities such as HOD and Warden to review and manage requests efficiently.

---

## 📌 Features

* 👩‍🎓 Student leave application system
* 🧑‍💼 Role-based access (Student, HOD, Warden)
* ✅ Leave approval/rejection workflow
* 📂 Organized data models for different users
* ⚡ Simple and clean backend structure using Node.js

---

## 🛠️ Tech Stack

* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Other Tools:**

  * JavaScript
  * npm

## 🚀 Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/FizaShaikh293/web-development-projects.git
```

### 2️⃣ Navigate to project folder

```bash
cd web-development-projects/LeaveManagement-NodeJS
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Run the application

```bash
node app.js
```

---

## 🔑 How It Works

1. Students submit leave requests
2. Requests are processed through the system
3. HOD and Warden can:

   * Approve ✅
   * Reject ❌
4. Status is updated accordingly
